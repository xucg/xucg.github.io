这是一篇示例文章，用来测试发帖
行内公式示例：直角三角形满足 \(a^2 + b^2 = c^2\)。

块级公式示例：

\[
c = \sqrt{a^2 + b^2}
\]\[
c = \sqrt{a^2 + b^2 }
\]

也可以写多行公式：

\[
\begin{aligned}
a^2 + b^2 &= c^2 \\
c &= \sqrt{a^2 + b^2}
\end{aligned}
\]
